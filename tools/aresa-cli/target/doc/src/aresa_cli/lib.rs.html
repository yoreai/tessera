1//! ARESA CLI - Natural Language Data Search
2//!
3//! A high-performance CLI tool for searching filesystems, databases,
4//! and cloud storage using natural language queries.
5//!
6//! # Features
7//!
8//! - **Natural Language Interface**: Ask questions in plain English
9//! - **Universal Search**: Files, PostgreSQL, BigQuery, S3, GCS, SQLite
10//! - **Beautiful Output**: Rich terminal formatting with tables and colors
11//! - **Secure**: Credentials stored in OS keychain
12//! - **Fast**: Built in Rust with async I/O
13//!
14//! # Example
15//!
16//! ```bash
17//! # Search files
18//! aresa "find python files with TODO comments in ~/dev"
19//!
20//! # Query database
21//! aresa "show me users who signed up last week"
22//!
23//! # Configure sources
24//! aresa config add postgres prod --uri postgresql://...
25//! ```
